🚀 Quick Start (User)

  1. Open any website page
  2. Find globe icon (🌐) in top navigation
  3. Click to open language menu
  4. Select:
    🇬🇧 English or
    🇮🇳 हिंदी (Hindi)
  5. Content updates instantly ✓
  6. Choice saved automatically

🔧 Developer API

Core Methods

➕ Add New Language (e.g., Spanish)

Step 1: Edit i18n.js

es: {
 'nav.home': 'Inicio',
 // Add all keys
}

Step 2: Update Switcher

'es': {
 flag: '🇪🇸',
 name: 'Español'
}

Step 3: Test

Open any page, click language button, verify Spanish content appears.

📝 Naming Best Practices

✅ Correct Format

  • nav.home
  • hospitals.title
  • register.email
  • msg.success

❌ Avoid

  • NavHome (use lowercase)
  • nav_home (use dots)
  • navHome (use dots)

🔍 Troubleshooting

Language Not Switching?

  • Check DevTools → Console for errors
  • Verify scripts are loaded (Network tab)
  • Clear cache and refresh page

Translations Missing?

  • Check data-translate attribute spelling
  • Verify key exists in translations object
  • Ensure DOM content exists on load
